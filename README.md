# Sophia-Token
A practice ERC-20 token (Sophia Token / SPT) built on Ethereum with a total supply of 1,000,000 SPT. Developed as part of the TechCrush Web3 Cohort7 Bootcamp to explore Solidity smart contract development, token standards, and blockchain fundamentals.

## Features
- Mintable
- Pausable
- Burnable
- ERC-20 Standard

## Tech Stack
- Solidity
- Foundry
- OpenZeppelin
